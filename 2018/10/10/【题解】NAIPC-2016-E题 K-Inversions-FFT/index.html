E. K-Inversions

题目链接：https://nanti.jisuanke.com/t/32229

题目原文

You are given a string s consisting only of upper case letters A and B. For an integer k, a pair of indices i and j(1≤i<j≤n) is called a k-inversion if and only if s[i]=B, s[j] = A and j - i = k.

Consider the string BABABABA. It has two 1-inversions and one 3-inversion. It has no 2-inversions.

For each k between 1 and n - 1 (inclusive), print the number of k-inversions in the string s.

Input

Each input will consist of a single test case. Note that your program may be run multiple times on different inputs. The input will consist of a single line with a string ss, which consists of only upper case As and Bs. The string s will be between 1 and 1,000,000 characters long. There will be no spaces.

Output

Output n−1 lines, each with a single integer. The first line’s integer should be the number of 1-inversions, the second should be the number of 2-inversions, and so on.

样例输出1

样例输入2

样例输出2

题意

给定一个只含AB两种字符的串，如果(i,j)满足i≤j且s[i]=B, s[j] = A，j-i = k，则(i,j)满足k-inversion。对于所有k，求满足k-inversion的总对数。

解析

考虑构造一个多项式乘法，多项F描述A，另一个多项式G描述B。

对于F，第 j 个位置的A即为 j 次方项，系数1，其它项系数为0

对于G，第 i 个位置的B即为 (-i) 次方项，系数1，其它系数为0

F与G相乘后，k次方项，即 ( j + (-i) ) 次方项系数即为K-Inversions的答案。

为了进行方便进行FFT，可以将G的每个位置+n，变为正数次方项，结果为[n,n+n-1]项的系数。
